자바스크립트란?

웹 브라우저에서 사용되는 인터프리터 방식의 스크립트 언어. 브라우저에 js엔진인 V8이 탑제되어 있어, 추가적인 설치없이 실행이 가능. * 인터프리터 방식 : 코드를 한줄씩 읽어가며 실행하는 방식

js를 사용하는 방법

  1. inline(인라인) : 태그 내 간단하게 코드를 직접 작성.
  2. internal(내부 참조) : html문서 내에 소스코드를 작성.
  3. external(외부 참조) : 별도의 .js파일로 소스코드를 작성.

1) inline 방식

태그의 on이벤트명="js코드"



2) internal 방식

script태그 영역안에 소스코드를 작성
script태그는 head태그, body태그 어디에서 건 사용해도 무관

3) external 방식

외부의 js파일을 생성해서 따로 js코드를 관리한다.
